
# 🕳️ Drilling Process (FlatCAM)

## Steps:
1. Load the **drill file** (`.drl`) into FlatCAM
2. Set drill parameters:
   - Tool Diameter: depends on your bit (`0.8 mm`, `1 mm`, etc.)
   - Feedrate: `150 mm/min`
   - Drill Depth: `-1.45 mm`
   - Dwell Time: `0.5 s`

3. Generate G-code and save as `drill.nc`
